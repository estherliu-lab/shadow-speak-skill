最后才考虑 YouTube

YouTube Is Optional

YouTube は最後の選択肢

只复制 YouTube 链接并不稳定。最稳方法是先复制字幕文字，或准备 SRT/VTT 文件，再交给 Skill 生成学习包。

A bare YouTube link is not reliable. The stable path is to copy the transcript text or prepare an SRT/VTT file, then give that text to the skill.

YouTube リンクだけでは安定しません。安定する方法は、字幕テキストをコピーするか SRT/VTT を用意してから skill に渡すことです。

复制字幕文字

Copy Transcript Text

字幕テキストをコピー

打开视频，点标题或简介附近的 ...、更多 或 Show transcript / 显示文字稿，打开后选中文字并复制，再贴给 Codex。

Open the video, choose ..., More, or Show transcript near the title or description, select the transcript text, copy it, and paste it into Codex.

動画を開き、タイトルや概要欄付近の ...、More、または Show transcript を開き、文字を選択してコピーし、Codex に貼ります。

上传或粘贴 SRT/VTT

Use SRT/VTT

SRT/VTT を使う

如果你已经有字幕文件，直接上传或把内容粘贴出来。Skill 会先清理时间轴、编号和重复行，再生成学习包。

If you already have a caption file, upload it or paste the content. The skill cleans timestamps, cue IDs, and repeated lines before generating the study pack.

字幕ファイルがある場合は、アップロードするか内容を貼ってください。タイムスタンプや重複行を整理してから教材化します。

什么时候只贴链接

When Links Work

リンクだけで使える場合

只有公开视频暴露了公开字幕轨时，才可以尝试复制链接检测一次。复制方法：点 YouTube 的 Share / 分享，再点 Copy / 复制链接。如果没有公开字幕轨，只贴链接无法生成学习包。

A link only works when the public video exposes a public caption track. Copy it with Share then Copy. If no public caption track exists, the link alone cannot create a study pack.

公開動画に公開字幕トラックがある場合だけ、リンクで一度試せます。Share から Copy します。公開字幕がなければ、リンクだけでは教材を作れません。
