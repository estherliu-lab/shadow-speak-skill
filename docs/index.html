YouTube 读取边界

YouTube Transcript Boundaries

YouTube 取得の範囲

这个 skill 只尝试公开可访问的字幕或页面可见文稿。无法访问时，它会请你粘贴字幕，不会假装读到了视频内容。

The skill only tries publicly accessible captions or visible transcripts. If unavailable, it asks you to paste captions instead of inventing content.

公開・表示可能な字幕だけを試します。取得できない場合は、内容を作らず字幕の貼り付けを依頼します。

python scripts/youtube_transcript_fetcher.py "https://www.youtube.com/watch?v=VIDEO_ID" transcript.txt
python scripts/subtitle_cleaner.py transcript.vtt cleaned.txt
python scripts/anki_csv_exporter.py cards.json cards.csv
