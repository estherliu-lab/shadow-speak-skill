## 001 医院说明孩子发烧

**适合场景：** 医院说明孩子发烧 时，先用一句简洁日语说明重点，再补充细节。

**先这样说：** すみません、少し確認したいことがあります。

**真实对话：**

A: すみません、少しお時間よろしいでしょうか。  
B: はい、大丈夫です。  
A: 念のため確認させてください。  
B: わかりました。状況を確認しますね。

**中文意思：** 不好意思，可以占用您一点时间吗？我想确认 / 咨询 / 更改 / 请教一下。

**关键表达：**

- 少しお時間よろしいでしょうか - 可以占用您一点时间吗
- 念のため確認させてください - 我想保险起见确认一下
- ご相談したいです - 想和您商量一下

**更礼貌一点：** お忙しいところ恐れ入りますが、少しご相談させていただけますでしょうか。

**跟读练习：** すみません / 少しお時間よろしいでしょうか / 念のため確認させてください

**替换练习：** 把“確認”替换成“変更 / 相談 / 予約 / 説明”。
