# Naturalness Check Rules / 语感校准案例库

这些案例用于训练“语法正确”和“听起来自然”之间的差异。所有例句为原创示例。

## 001 语气偏硬

**原句：** 私はあなたに会う時間がありません。

**判断：** 可以理解，但语气偏硬，真实沟通里建议稍微调整。

**更自然：** 最近少し忙しくて、なかなかお会いする時間が取れなくて。

**口语一点：** 最近少し忙しくて、なかなかお会いする時間が取れなくて。

**礼貌一点：** 最近少し忙しくて、なかなかお会いする時間が取れなくて。

**说明：** 直接说“没有时间见你”容易显得拒绝过硬。

**替换练习：** 把同样语气用于“医院说明孩子发烧”这个场景。

## 002 有翻译腔

**原句：** 私はとても疲れた気がします。

**判断：** 可以理解，但有翻译腔，真实沟通里建议稍微调整。

**更自然：** 最近ちょっと疲れ気味です。

**口语一点：** 最近ちょっと疲れ気味。

**礼貌一点：** 最近ちょっと疲れ気味ですので、少しご相談できますでしょうか。

**说明：** 「疲れた気がします」像中文直译，日常更常说「疲れ気味」。

**替换练习：** 把同样语气用于“医院说明喉咙痛”这个场景。
