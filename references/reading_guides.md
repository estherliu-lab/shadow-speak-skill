# Reading And Shadowing Guides

## English

- Mark short pauses with `/`.
- Mark sentence stress with `**bold**`.
- Point out reduced speech only when it helps speaking, such as "going to" -> "gonna" in casual contexts.
- Explain tone with practical labels: warm, surprised, doubtful, joking, firm, nostalgic, hesitant.
- For shadowing, give three speeds: slow chunking, natural sentence, emotional imitation.

Example:

```markdown
Original: I did not realize how much I missed this place.
Slow: I did not **realize** / how much I **missed** / this **place**.
Tone: nostalgic, with a small moment of surprise.
Task: Read slowly 3 times, natural speed 3 times, then retell the feeling without looking.
```

## Japanese

- Mark bunsetsu or meaning chunks with `/`.
- Add kana reading for kanji-heavy sentences when useful.
- Explain falling tone, surprise, hesitation, politeness, softness, and implied refusal.
- For long subtitles, choose sentences that contain reusable grammar or natural conversation.

Example:

```markdown
原句: まさか、こんなところで会えるとは思わなかった。
かな: まさか、こんなところで あえるとは おもわなかった。
切分: まさか / こんなところで / 会えるとは / 思わなかった。
语气: 前半有惊讶，后半自然下降，像是真的没想到。
```

## Retelling

Always include one small output task:

- Beginner: repeat with one word changed.
- Intermediate: retell the sentence in simpler language.
- Advanced: use the expression in a new scene.
