# Creator Formats

## X 短帖

**模板：** 中文一句真实感受 + 日语自然表达 + 一个语感注释。

**示例：** 最近有点累，但还是想继续。 / 最近少し疲れ気味だけど、もう少し続けてみたい。

**注意事项：** 保持像真人会发出的语气，不堆砌漂亮词。

**不要这样写：** 不要写成夸张鸡汤或机器翻译式长句。
