# Examples

## English Sentence Breakdown

User:

```text
I am not sure if that is a good idea.
```

Output focus:

- Explain that it is a soft disagreement, not simple uncertainty.
- Offer alternatives: "I have some concerns", "I am not fully convinced", "Maybe we should think it through."
- Add shadowing stress on `not sure` and `good idea`.

## Japanese Sentence Breakdown

User:

```text
それはちょっと難しいかもしれません。
```

Output focus:

- Explain that `ちょっと` softens refusal.
- Give alternatives by politeness level.
- Add kana and slow chunking.

## One Idea, Many Uses

Chinese:

```text
我最近有点累，但还是想继续努力。
```

English:

- Casual: I have been kind of tired lately, but I still want to keep going.
- Social: Tired, yes. Done, no.

Japanese:

- 自然版: 最近少し疲れているけど、もう少し頑張りたい。
- 温柔版: 少し疲れてはいるけれど、もう少しだけ前に進みたい。
