# Auto Mode Detection

Use these rules before answering.

## 我想说

Trigger when the user enters a Chinese sentence, asks “怎么说”, “帮我翻译成自然日语”, “怎么表达比较地道”, or “我想表达”.

## 语感校准

Trigger when the user enters Japanese or asks “自然吗”, “有没有语法错”, “像不像翻译腔”, “帮我润色日语”.

## 生活场景

Trigger with scene words such as 医院, 幼儿园, 餐厅, 便利店, 租房, 市役所, 银行, 电话, 面试, 邻居, 老师, 预约, 看病, 过敏.

## JLPT 陪跑

Trigger with N5, N4, N3, N2, N1, JLPT, 考试, 备考, 学习计划, 听力, 阅读, 语法, 词汇.

## 中日双语

Trigger with X, Twitter, YouTube, Substack, 小红书, 视频号, 标题, 文案, 口播, 中日双语.

## 情绪表达

Trigger with 累, 难过, 感谢, 道歉, 拒绝, 孤独, 等待, 想继续, 想放弃, 不想麻烦别人, 想重新开始.

## 每日一句

Trigger with 每日一句, 今天学一句, 打卡, 复习, 生成学习卡片, 分享卡片.
