# 日语开口急救包 / Nihongo Mouthpiece

把中文想法，变成自然日语。

你不是不会日语，你只是一直在说“翻译腔日语”。这个 Skill 帮中文母语者把脑子里的真实想法，变成更自然、更柔和、更适合真实沟通的日语。

## 它是什么

这是一个零成本、可发布到 GitHub 的 Codex Skill 项目。它不是在线翻译器，不调用 API，不需要服务器，不上传用户数据。它是一套可以被 Codex 读取和执行的日语表达工作流，配套有参考库、示例库和一个单文件静态展示页。

它适合：

- 日语初学者
- 学过日语但不敢开口的人
- 写日语总像中文直译的人
- 正在准备 N5-N1 的学习者
- 想去日本生活、留学、工作、带娃的人
- 想做中日双语内容的人
- 想用日语表达情绪但怕尴尬的人
- 想每天打卡一句自然日语的人

## 七个模式

### 我想说

把中文想法变成自然、可直接说出口的日语。

### 语感校准

判断日语是否自然，修正翻译腔、语气硬和表达别扭。

### 生活场景

覆盖日本生活里的真实沟通场景，给句子、对话和替换练习。

### JLPT 陪跑

把备考拆成每天能完成的小任务。

### 中日双语

把内容变成适合发布的中日双语文案。

### 情绪表达

把感谢、疲惫、拒绝、抱歉说得自然且不尴尬。

### 每日一句

每天记住一句能用出去的日语，并生成打卡内容。

## 为什么它有用

普通翻译经常只回答“这句话日语怎么说”。但真实沟通里，更重要的问题是：

- 这句话对朋友说会不会太硬？
- 给老师或上司会不会不够礼貌？
- 语法没错但听起来像不像中文直译？
- 对方能不能感受到我的委婉、感谢或顾虑？
- 我能不能把这句话明天真的说出口？

日语开口急救包把答案拆成几个可直接使用的版本：自然版、口语版、礼貌版、委婉版、可打卡版。用户不用一次学很多，只要先拿到一句能说出口的话。

## 如何安装为 Codex Skill

把 `nihongo-mouthpiece` 文件夹放到你的 Codex skills 目录，例如：

```text
C:\Users\你的用户名\.codex\skills\nihongo-mouthpiece
```

重启或刷新 Codex 后，当用户提出中文转自然日语、语感校准、生活场景日语、JLPT 学习、每日一句等请求时，这个 Skill 就可以被触发。

## 如何预览展示页

直接打开：

```text
docs/index.html
```

它是纯 HTML、内联 CSS、原生 JavaScript，没有 React、Vue、Tailwind、CDN、字体、图片、API 或后端。

## 如何发布到 GitHub Pages

1. 把项目推到 GitHub。
2. 打开仓库 Settings。
3. 进入 Pages。
4. Source 选择 `Deploy from a branch`。
5. Branch 选择 `main`，目录选择 `/docs`。
6. 保存后等待 GitHub Pages 生成链接。

发布后，你可以把链接生成二维码。手机扫码就是一个接近 App 的体验，也可以“添加到主屏幕”。

## 零成本说明

这个项目本体不产生额外费用，因为它不使用：

- OpenAI API
- Claude API
- Gemini API
- 词典 API
- 数据库
- 登录系统
- 后端服务器
- 外部字体
- 外部图片
- 付费模板

用户真正使用 AI 生成内容时，只消耗用户自己在 Codex 或其他 AI 工具里的额度。

## 设计方向

界面参考日系手账、奶油白、抹茶绿、浅咖啡、柔和橘粉。它应该像随身日语嘴替：温柔但不空，实用但不硬，有趣但不幼稚，内容丰富但不臃肿。
