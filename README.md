## 这是做什么的

日语开口急救包适合已经学过一点日语、但开口时总觉得不自然的人。

你可以把中文想法放进去，参考它给出的自然日语表达、语感说明、替换练习和每日一句。它重点解决的不是“单词怎么翻译”，而是“这句话怎么说更像真实沟通里的日语”。

## 适合谁用

- 想把中文想法说成自然日语的人
- 写日语时容易有翻译腔的人
- 想练习日本生活场景表达的人
- 想每天积累一句可直接使用日语的人
- 想做中日双语内容的人
- 想把感谢、拒绝、抱歉、疲惫说得更柔和的人

## 怎么用

1. 扫二维码或打开上面的链接。
2. 在输入框里写一句中文想法，例如“最近有点累，但还是想继续努力”。
3. 查看自然日语版、口语版和更温柔的表达。
4. 复制你最想记住的一句，用在聊天、笔记或每日打卡里。

## 可以练什么

- 我想说：把中文想法变成自然日语
- 语感校准：看看一句日语哪里不自然
- 生活场景：医院、幼儿园、租房、餐厅、电话预约等
- 情绪表达：感谢、拒绝、道歉、疲惫、想重新开始
- 每日一句：每天记住一句能用出去的日语

## 今日一句

日语：

> 最近少し疲れ気味だけど、もう少しだけ続けてみたい。

中文：

> 最近有点累，但还是想再坚持一下。

语感：

> 「疲れ気味」比直接说「疲れている」更柔和，适合表达“有点累，但还想继续”的状态。
